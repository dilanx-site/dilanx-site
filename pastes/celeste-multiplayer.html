Celeste Multiplayer Installation Guide

            1. Install Everest, the Celeste Mod Loader, using Olympus here.
            2. Open Olympus, go to Installations and press Manage, and select your installation of Celeste (if it isn't selected already by default).
            3. Press Download Mods from the main menu, search for CelesteNet, and download it.
            4. Press Everest from the main menu to launch modded Celeste. Keep it running.
            5. Go to the CelesteNet website, scroll to the bottom, and link your Discord. Then, press Send to Client.
